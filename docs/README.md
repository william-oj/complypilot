# ComplyPilot

ComplyPilot is a production-ready ISO/IEC 27001 automation platform that turns complex compliance into a guided, step-by-step journey. The system includes a full backend, frontend, AI assistant, policy generator, and deployment stack.

## Workspace layout
- `backend` Node.js + Express + Prisma API
- `frontend` Next.js UI
- `infra` Docker, compose, CI/CD
- `docs` Architecture, API, security, deployment

## Quick start (local)
1. Start dependencies using Docker Compose.
2. Install backend and frontend dependencies.
3. Run database migrations and seed.
4. Start API and UI.

See `docs/DEPLOYMENT.md` for exact commands.
